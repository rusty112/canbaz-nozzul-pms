# CANBAZ NOZZUL PMS v3.0

## Overview
CANBAZ NOZZUL PMS is a modern, web-based Planned Maintenance System designed for maritime operations. It features a responsive dashboard, department-specific maintenance trees (Engine & Deck), smart compliance tracking, and RightShip integration.

## Features
- **Dashboard:** Real-time overview of vessel status, overdue items, and compliance.
- **Smart Compliance:** Automated generating of maintenance dates to ensure compliant status.
- **Departments:**
  - **Engine:** Heirarchical tree view of components with running hours and maintenance intervals.
  - **Deck:** Standard checklist-based maintenance tracking (GV-05-A).
- **RightShip:** Integration of RightShip inspection checklist.
- **Role-Based Access:** Logins for Master, C/E, C/O, 2/E, etc. with specific views.

## Installation
1. Ensure Node.js is installed.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.

## Technologies
